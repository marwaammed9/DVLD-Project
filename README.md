# 🪪 DVLD - Driving License Management System

A full-scale, real-world Driving License Management System 
built with C# Windows Forms and SQL Server.

## 📌 About The Project
This system was built as part of a complete backend development 
roadmap, applying all concepts learned including OOP, Clean 
Architecture, ADO.NET, and SQL Server.

## ✨ Features
- 👤 People & Users Management
- 🔐 Login and Permissions System
- 📋 License Applications Management
- 🪪 Driving License Issuance & Renewal
- 🔄 Replacement for Lost or Damaged Licenses
- ⛓️ Detain & Release Licenses
- 🌍 Local and International Driving Licenses
- 📊 Reports and History Tracking
- 🧪 Test Types and Application Types Management

## 🛠️ Tech Stack
- C# / .NET Windows Forms
- SQL Server
- ADO.NET
- 3-Layer Architecture (DAL / BLL / UI)

## 🏗️ Architecture
- **DataAccessDVLD** — Data Access Layer (DAL)
- **BusnissDVLD** — Business Logic Layer (BLL)
- **DVLD Project** — Presentation Layer (UI)

## 🗄️ Database
See the `/Database` folder for the full SQL script.
